# CSV & XML to JSON Converter
Утилита для преобразования XML-файлов в JSON-формат. Проект демонстрирует работу с DOM-парсером и сериализацией данных в Java.

### О проекте
Программа читает XML-файл со структурой данных сотрудников, парсит его и сохраняет содержимое в JSON-файл. Реализована в рамках учебного курса по Java-разработке.

### Функционал
Парсинг XML-документов с использованием DocumentBuilderFactory

Извлечение данных из узлов NodeList и Element

Создание объектов Employee из XML

Преобразование списка объектов в JSON

Запись результата в файл
